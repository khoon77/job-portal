\26-3기 임관시 장기복무 부사관 모집 공고 입니다.

* 지원서 접수는 국방통합모집포털(https://m-recruit.mnd.go.kr/) 회원등록(로그인) 후 가능합니다.

□ 회원등록 절차

1. 홈페이지 접속 후

2. 좌측 상단 로그인 버튼 클릭

3. 우측 하단 회원가입 클릭

4. 회원가입

□ 지원서 접수 절차

1. 로그인

2. 메인 홈페이지에서 지원과정(26-3기 임관시 장기복무 부사관 모집 공고) 클릭

3. 지원과정 공지문 맨 하단 지원하기 버튼 클릭

4. 지원서 작성

※ 세부사항은 국방통합모집포털 내 육군 &gt; 지원/접수 &gt; 공고문 확인 바랍니다.
