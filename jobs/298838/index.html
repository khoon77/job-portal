26-4기 민간부사관(남,여군) 모집 공고 입니다.

* 지원서 접수는 회원등록(로그인) 후 가능합니다.

□ 주요 변경사항

1. 지원서 접수 시 희망특기 선택 → 전 병과(특기) 통합지원

단, 군악특기는 기존과 동일하게 최초 지원서 접수 시 1지망으로 지원 가능

* 기존(지망선택) : 1지망 (전투특기), 2지망 (인사), 3지망 (조리)

* 변경(통합선택) : 1지망 (전 병과 통합) or 군악특기

2. 특기분류 : 선발 이후 부사관학교 양성교육 중 실시

* 부사관학교 양성교육 7~9주차 특기분류 (양성교육 12주 기준, 일정 변동가능)

* 특기부여 기준 : 전공경력, 자격 / 면허, 양성교육 성적 점수 등을 합산하여

고득점자를 희망특기에 우선 분류

□ 회원등록 절차

1. 홈페이지 접속 후

2. 좌측 상단 로그인 버튼 클릭

3. 우측 하단 회원가입 클릭

4. 회원가입

□ 지원서 접수 절차

1. 로그인

2. 메인 홈페이지에서 지원과정(26-4기 민간부사관(남, 여 군) 모집 공고) 클릭

3. 지원과정 공지문 맨 하단 지원하기 버튼 클릭

4. 지원서 작성

※ 세부사항은 국방통합모집포털 상 육군 &gt; 지원/접수 &gt; 공고문 확인 바랍니다.

(기타 문의: 042-550-7156(남군), 7155(여군))
